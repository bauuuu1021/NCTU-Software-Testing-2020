# Lab6 of NCTU SW Testing
a practice of generating call graph and control flow graph

## Usage
* Generate call graph
	```
	./call_graph.sh <C File> <target png file>
	```
* Generate control flow graph
	```
	./ctrl_flow.sh <C File>
	```
